# ytmark

ytmark is a YouTube bookmarking application that helps users save timestamps in videos and continue watching from where they left off.

## Features

- Save bookmarks instantly using the keyboard shortcut:
  
  `Alt + B`

- Automatically stores the current timestamp of the video.

- Displays a popup when you reopen the same video, allowing you to resume from your last saved timestamp.

- Useful for:
  - Long tutorials
  - Podcasts
  - Courses
  - Interviews
  - Any lengthy YouTube content

## How It Works

1. Open any YouTube video.
2. Press `Alt + B` whenever you want to save your current position.
3. The timestamp will be stored automatically.
4. When you revisit the same video, ytmark will show a popup asking if you want to resume from the saved timestamp.

## Why ytmark?

Watching long videos can be frustrating when you forget where you stopped. ytmark makes it easy to continue watching without manually searching for the previous timestamp.
